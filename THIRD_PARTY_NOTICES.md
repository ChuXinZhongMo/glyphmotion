# Third-Party Notices

GlyphMotion can bundle FFmpeg in Windows installer builds so video input and MP4 export work without a separate FFmpeg install.

FFmpeg is a separate executable maintained by the FFmpeg project. Windows installer builds use the Gyan.dev FFmpeg build when it is available through WinGet. The bundled FFmpeg binary keeps its own license terms, copyright notices, and source availability requirements.

- FFmpeg project: https://ffmpeg.org/
- FFmpeg legal information: https://ffmpeg.org/legal.html
